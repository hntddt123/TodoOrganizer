## Setup

To run development server 
```
npm run startDev
```

To Connect MongoDb for development 
MacOS
```
brew services start mongodb-community
```

Windows
https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/

To run Build
```
npm run build
```
